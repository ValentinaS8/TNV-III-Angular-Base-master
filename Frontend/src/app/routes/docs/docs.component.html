Our web application presents two screens: “Dashboard” and “Graphics”. It also has a working login / register system.

Login/Register

The login/register system doesn't save any data on DB, so, the system will keep track of your registering data only up to the next web application reboot. In order to access the dashboard or the charts, you have to register yourself. When you'll try to reach one of them, the system will ask you for your credentials, and it'll let you access the dashboard or the charts only if your data are correct.

Dashboard

Our Dashboard allows you to be aware of European weather and covid real-time data, according to the last updates from APIs.

Charts
Charts allow you to view the European data in a graph. The screen is splitted into two parts: in the first part you can see, in real-time, covid data and weather information concerning European countries, divided by types. Whereas, in the second part, you can find analysis based on the first part's data, as the ratio between deaths/cases and AQI (Air Quality Index), or the amount of deaths/cases in relation to the wind speed.
