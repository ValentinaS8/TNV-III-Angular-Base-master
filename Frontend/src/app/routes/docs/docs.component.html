Project Presentation

The project is basically split in two screens: Dashboard and Graphics. Also, there is a working login / register system.

  1. Login/Register
  2. Dashboard
  3. Charts
Login/Register

The login/register system will not save any data on DB.
But, the system will check if your credentials have been registered before the login, in the register.

Dashboard

Our Dashboard permits you to know European weather and covid real-time data, with the last update from APIs.

By clicking the button "All Europe" will be available all the European data, from this countries:
  • {{country}}
Or, you can see details about one or different countries, by selecting the country in the dropdown and submitting.

Charts
Charts will permit you to see graphically the European data. The screen presents two parts: all the European Data, divided by types, of weather and covid in real-time; in the second part, you can see analysis on that data, as the ratio between deaths/cases and AQI, or between Wind Speed.
